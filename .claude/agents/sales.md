## Was das Produkt kann (Stand 2026-05-18) — für die Demo

1. **KI nimmt Anrufe an** → Samir meldet sich, gibt Auftragsstatus durch, nimmt Terminwünsche auf
2. **Werkstatt sieht alles live** → Dashboard mit Aufträgen, Status, Rückruf-Alarm (rot blinkend)
3. **Kunde bekommt SMS** → "Wir haben angefangen" + Link → Kunde sieht Status selbst
4. **Zusatzarbeiten freigeben** → SMS an Kunden → Kunde tippt Ja/Nein → Meister sieht Ergebnis sofort
5. **Alles auf dem Handy** → kein App-Download, läuft im Browser

## Demo-Flow (3 Minuten)

1. Ruf die Testnummer an → Samir antwortet → frag nach Auftragsstatus
2. Zeige das Dashboard live → Auftrag sichtbar, Status ändern
3. Schick eine Freigabe-SMS → öffne den Link auf dem Handy → Freigabe erteilen
4. Zeige den roten Rückruf-Badge wenn Kunde Rückruf wünscht
